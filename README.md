# Warehouse Management System Frontend

This Angular repository provides the user interface for the warehouse management system.

## Initial Frontend Scope

- application shell with navigation
- dedicated pages for dashboard, suppliers, inventory, orders, and finance
- route structure that mirrors the assignment feature areas
- static content for the first increment so design and workflow structure can be reviewed early

## Next Frontend Steps

1. Add typed API service classes.
2. Replace static placeholder content with live backend data.
3. Add reactive forms for supplier, inventory, and order workflows.
4. Add component and service tests.
